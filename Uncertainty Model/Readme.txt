Description of the data:
Dune1, Dune2, Dune3 - Simulated coordinates of the profile of the sea-bottom
rho1, rho2, rho3 - Simulated angles of the survey,
m files
MAIN_CODE - MAin code with equations to be executed. 
Other files are functions created to the matin code. 
The code plot_gaussian_ellipsoid, show2s, show3d, axis and grid - used to display the uncertainty ellipsoids were not developed by the authors. The author is mentioned in plot_gaussian_ellipsoid
The code findPointNormals_mg - used to estimate the normal of the points was not made by the authors. The author is mentioned in the file.
